# CodeAlpha Task 1 - Web Scraping

## Project Overview

This project is developed as part of the CodeAlpha Data Analytics Internship.

The objective is to scrape book information from the "Books To Scrape" website using Python and BeautifulSoup.

The scraped data is stored in a CSV file for further analysis.


## Technologies Used

- Python
- Requests
- BeautifulSoup4
- Pandas

## Features

- Scrapes data from the first 10 pages.
- Extracts Book Title.
- Extracts Book Price.
- Stores all records in one CSV file.
- Simple and clean Python code.
- Beginner friendly.

## Dataset

Website: https://books.toscrape.com/

Output File: books_dataset.csv
