BOOK PRICE EDA PROJECT

PROJECT OVERVIEW
This project performs Exploratory Data Analysis (EDA) on a dataset of books to understand price patterns and basic statistics.

DATASET
- 200 rows, 2 columns
- Columns: Title, Price
